## 🔍 Project Overview

This project answers a core marketing analytics question:

> *When TV, Radio, Social Media spend, and Influencer type are all considered simultaneously, which channels have a statistically significant and independently measurable effect on Sales? How should the marketing budget be prioritised?*

Unlike simple regression, Multiple Linear Regression (MLR) controls for all predictors simultaneously, enabling each coefficient to be interpreted **holding all other variables constant**. This removes confounding and produces cleaner, more actionable business insights.

The analysis pipeline covers:
- Categorical variable encoding (ordinal and one-hot)
- Multicollinearity diagnosis via correlation matrices and Variance Inflation Factors (VIF)
- Full OLS model fitting using `statsmodels`
- Backward elimination based on p-values and Adjusted R-squared
- Residual diagnostic validation (Linearity, Normality, Homoscedasticity, Independence)
- Stakeholder-friendly coefficient interpretation
- Evidence-based budget allocation recommendation

## 🧪 Methodology

### Categorical Encoding

Two different encoding strategies are applied based on the nature of each categorical variable:

**TV — Ordinal Encoding**
TV budget levels have a natural, meaningful order (Low < Medium < High). Preserving this order via integer encoding (1, 2, 3) is more appropriate than one-hot encoding, which would treat the levels as unordered and add unnecessary dimensionality.

| TV Level | Encoded |
|---|---|
| Low | 1 |
| Medium | 2 |
| High | 3 |

**Influencer — One-Hot Encoding (Reference: Macro)**
Influencer type has no natural ordering. We create binary indicator variables for Mega, Micro, and Nano, dropping Macro as the reference category to avoid the dummy variable trap (perfect multicollinearity in the design matrix).

Each dummy coefficient then represents the Sales difference relative to a Macro influencer campaign, all else equal.

---

### Multicollinearity Detection

#### 1. Correlation Matrix
Pearson correlations among numeric predictors:

| | TV (encoded) | Radio | Social Media |
|---|---|---|---|
| **TV (encoded)** | 1.000 | 0.803 | 0.512 |
| **Radio** | 0.803 | 1.000 | 0.630 |
| **Social Media** | 0.512 | 0.630 | 1.000 |

The TV–Radio correlation (r = 0.803) is high and warrants VIF analysis.

#### 2. Variance Inflation Factor (VIF)

$$\text{VIF}_j = \frac{1}{1 - R_j^2}$$

where $R_j^2$ is the R-squared from regressing predictor $j$ on all remaining predictors.

| Feature | VIF | Status |
|---|---|---|
| Radio | 3.47 | Moderate — acceptable |
| TV_encoded | 2.83 | Moderate — acceptable |
| Social Media | 1.67 | OK |
| Influencer_Nano | 1.63 | OK |
| Influencer_Micro | 1.62 | OK |
| Influencer_Mega | 1.59 | OK |

All VIF values are below 5. While TV and Radio exhibit moderate collinearity (reflecting their r=0.80 correlation), the VIFs remain well within the acceptable threshold. No remediation (e.g. Ridge regression, variable removal) is required.

---

### Model Building: Backward Elimination

**Step 1 — Full Model (6 predictors)**

All encoded predictors are included first. Results:

| Predictor | Coefficient | p-value | Significant? |
|---|---|---|---|
| Intercept | -13.91 | 0.0002 | Yes |
| TV_encoded | 77.45 | < 0.0001 | **Yes** |
| Radio | 2.96 | < 0.0001 | **Yes** |
| Social Media | -0.15 | 0.828 | No |
| Influencer_Mega | 2.62 | 0.449 | No |
| Influencer_Micro | 2.98 | 0.378 | No |
| Influencer_Nano | 0.74 | 0.824 | No |

**Step 2 — Variable Removal Decision**

Social Media and all three Influencer dummies have p > 0.05. They are dropped together, and the Adjusted R-squared is compared:

| Model | Predictors | Adj. R-squared | RMSE |
|---|---|---|---|
| Full Model | TV, Radio, Social Media, 3 Influencer dummies | 0.9031 | 27.81 |
| **Reduced Model** | **TV, Radio** | **0.9036** | **27.83** |

The reduced model achieves a **higher** Adjusted R-squared, confirming the removed variables were adding noise rather than signal.

**Final Model:**
```
Sales = -12.39 + 77.32 × TV_encoded + 2.96 × Radio + ε
```

## ✅ Assumption Validation

### Results Summary

| Assumption | Test | Statistic | Result |
|---|---|---|---|
| **Linearity** | Residuals vs Fitted (visual) | Flat rolling mean | ✅ Pass |
| **Normality** | Shapiro-Wilk | p < 0.05 (large n effect) | ⚠️ See note |
| **Normality** | Jarque-Bera (from statsmodels) | p = 0.254 | ✅ Pass |
| **Homoscedasticity** | Breusch-Pagan | p = 0.760 | ✅ Pass |
| **Independence** | Durbin-Watson | DW = 1.882 | ✅ Pass (near 2) |

### Normality Note
The Shapiro-Wilk test rejects normality (p < 0.05), but this is a known artefact of large samples (n = 572) where even trivial, practically irrelevant deviations from normality become statistically detectable. Visual inspection of the Q-Q plot shows points closely following the reference line; the residual histogram closely matches the Normal PDF. The Jarque-Bera test (p = 0.254) confirms there is no significant skewness or excess kurtosis. Given the sample size and the Central Limit Theorem, inference on regression coefficients remains fully valid.

## 💼 Coefficient Interpretation

The final model equation:

```
Sales ($K) = -12.39 + 77.32 × TV_encoded + 2.96 × Radio
```

**Intercept (β₀ = -12.39)**  
The mathematical baseline when all predictors equal zero. This lies outside the real data range (TV minimum is Low = 1) and should not be interpreted as a meaningful standalone business figure.

**TV_encoded (β₁ = 77.32, 95% CI: [72.50, 82.14])**  
Holding Radio spend constant, moving TV budget up by one tier (Low → Medium, or Medium → High) is associated with an average increase of **$77,320 in Sales**. This is the single most impactful lever in the model.

| TV Budget Change | Expected Sales Impact |
|---|---|
| Low → Medium | +$77.3K |
| Medium → High | +$77.3K |
| Low → High (2 tiers) | +$154.6K |

**Radio (β₂ = 2.96, 95% CI: [2.54, 3.37])**  
Holding TV tier constant, each additional **$1,000** invested in Radio advertising is associated with an average increase of **$2,957 in Sales** (ROAS: approximately 2.96×).

| Radio Investment | Expected Sales Impact |
|---|---|
| +$10K | +$29.6K |
| +$20K | +$59.1K |
| +$30K | +$88.7K |

**Social Media and Influencer Dummies**  
After controlling for TV and Radio, neither Social Media spend nor any Influencer category shows a statistically significant relationship with Sales. Their contribution is subsumed by the two primary predictors.

## 💼 Business Recommendation

### Prioritised Marketing Budget Allocation

Based on a statistically robust Multiple Linear Regression model (Adj. R² = 0.9036, n = 572):

**Priority 1 — TV Advertising (Highest Impact)**  
Upgrade TV budget tier as the first and most impactful investment. A single-tier upgrade from Low to Medium or Medium to High is associated with +$77.3K in Sales. Maximising TV spend to the High tier should be the primary budget objective.

**Priority 2 — Radio Advertising (Reliable ROAS)**  
Radio delivers a consistent, scalable return of approximately $2.96 for every $1 spent (ROAS: 2.96×), independent of TV tier. It should be maintained and grown as a secondary channel within budget constraints.

**Priority 3 — Social Media (No Detected Sales Impact — Evaluate Separately)**  
Social Media spend shows no statistically significant effect on short-term Sales after controlling for TV and Radio. This does not disprove its long-term brand or engagement value, but it cannot be justified on short-term Sales ROI grounds without further evidence. Recommendation: commission a dedicated A/B test or attribution model for Social Media.

**Priority 4 — Influencer Marketing (No Tier Differentiation Detected)**  
No Influencer category (Mega, Micro, or Nano) outperforms the Macro baseline in a statistically significant way. Influencer selection should be guided by other KPIs (engagement, reach, brand alignment) rather than short-term Sales lift.

### Suggested Budget Split

| Channel | Recommended Share | Rationale |
|---|---|---|
| TV (High tier) | 65–70% | Dominant revenue driver; largest single effect |
| Radio | 25–30% | Reliable ROAS of ~2.96×; scales linearly |
| Social Media | ≤ 5% (test budget) | No significant Sales impact detected |
| Influencer | Evaluate separately | No tier advantage in Sales data |

> **Important Caveat:** This model captures linear, short-term, channel-level effects. Marketing synergies (e.g. TV × Radio interactions), lagged effects, diminishing returns at high spend, and long-term brand equity are not modelled here. A more advanced model (e.g. multiplicative media mix model, Adstock modelling) would provide a more complete allocation picture.

**Refit with statsmodels:**

```python
import statsmodels.api as sm
import pandas as pd

df = pd.read_csv('marketing_sales_data.csv')
tv_map = {'Low': 1, 'Medium': 2, 'High': 3}
df['TV_encoded'] = df['TV'].map(tv_map)

X = sm.add_constant(df[['TV_encoded', 'Radio']])
y = df['Sales']
model = sm.OLS(y, X).fit()
print(model.summary())
```

## ⚠️ Limitations & Future Work

### Current Limitations

- **Ordinal TV encoding assumes equal intervals** between Low, Medium, and High. If the actual budget differences between tiers are unequal, this assumption introduces measurement error. One-hot encoding for TV (and including an F-test for the joint significance of the dummies) would be more rigorous.
- **No interaction terms** — the model does not test whether TV and Radio work synergistically. A TV × Radio interaction term may capture multiplicative effects.
- **Linearity assumed** — diminishing returns at high spend levels are not captured. A log or polynomial specification may improve fit at budget extremes.
- **No temporal dimension** — advertising effects often carry over multiple time periods (Adstock effect). Without a time index, lagged effects cannot be modelled.
- **Cross-sectional only** — the model treats all observations as independent. If the data contains repeated observations across time or market segments, a panel data or mixed-effects model may be more appropriate.
- **Social Media and Influencer exclusion** — statistical insignificance at this sample size does not prove zero effect. Brand equity, awareness, and long-term conversion effects are not captured in short-term Sales data.

### Suggested Extensions

- [ ] **Multiple regression with interaction terms** — e.g. `TV_encoded * Radio`
- [ ] **Log-linear transformation** — `log(Sales)` as the dependent variable to model proportional effects
- [ ] **Media Mix Modelling (MMM)** — incorporate Adstock, saturation, and time series effects
- [ ] **Ridge / Lasso Regression** — apply regularisation to manage the moderate TV–Radio collinearity
- [ ] **Train/test split + cross-validation** — add out-of-sample evaluation for generalisability
- [ ] **Segmented analysis** — run separate models by region, product, or season
- [ ] **One-hot TV encoding** — test whether the ordinal assumption holds via an F-test for TV dummies
- [ ] **Robust standard errors** — apply HC3 standard errors to guard against any residual heteroscedasticity
